# change state

Changes the global state. If the millis parameter is provided, the state will change after the given time instead of immediately.

```sig
stateTransitions.changeState("")
```

## Parameters

* **newState**: The state to change the global state to
* **millis**: The time to wait before changing the state, in milliseconds

```package
arcade-state-transitions=github:riknoll/arcade-state-transitions
```
